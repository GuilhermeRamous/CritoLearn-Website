Caesar Cipher

The cesar cipher is one of the oldest and most widely known cryptographic techniques in the world, being a cryptography based on replacing one letter of the alphabet with another using a key. The cryptographic technique is known as Caesar Cipher, since Julio Cesar (great politician and Roman military) used a cipher with exchange of three positions to protect military secrets.>

Operation:

To encrypt messages we will always need a common alphabet and an encrypted one that can be built with left-to-right or right-to-left rotation. To decrypt it is enough to make the substitution with the reverse rotation that was made to encrypt..

Example 1: common alphabet and encrypted alphabet of key 5 that rotates from left to right

ABCDEFGHIJKLMNOPQRSTUVWXYZ

FGHIJKLMNOPQRSTUVWXYZABCDE

Example 2: common alphabet and encrypted alphabet of key 5 that rotates from right to left

ABCDEFGHIJKLMNOPQRSTUVWXYZ

VWXYZABCDEFGHIJKLMNOPQRSTU

Note: Although there are two types of rotation, it is common to only make use of rotation from left to right. And as you can see, this does not affect encryption, the encryption of example 2 could be done using left-to-right rotation key 21.

Keyless decrypting:

We can decrypt cipher-encrypted messages using intuition and logical reasoning. All of this must be based on how words are formed in the language in which the original message was written.

Take the Portuguese language as an example, where words are formed by syllables, each syllable has a vowel (a, e, i, o, u) and each syllable usually has two or three letters. To practice, we will decrypt the JHZH message. But where can we begin?

If you count the number of letters you will see that there are four, in this way we can intuit that there is a great possibility that the word is formed by two syllables. If we separate the message into two syllables (JH-ZH) we will see that the letter H repeats, which gives us the clue that H is a vowel.

Since A is a letter that appears in many words of our language, I will kick that H is equivalent to A. If this is true, the key of the cipher in this case is 7. Decrypting the message using key 7, we call the following message : HOUSE.

Basically, when you do not know the key, you should decrease the number of attempts using the properties of the language in your favor.
