Caesar Cipher

The cesar cipher is one of the oldest and most widely known cryptographic techniques in the world, being a cryptography based on replacing a letter of the alphabet with another form of a key. Such a cryptographic technique is known as cesar cipher, since Julio Cesar (great politician and Roman military) uses a cipher with exchange of three positions to protect military secrets.

Operation

Example 1: common alphabet and encrypted alphabet of key 5 that rotates from left to right

ABCDEFGHIJKLMNOPQRSTUVWXYZ

FGHIJKLMNOPQRSTUVWXYZABCDE

Example 2: common alphabet and encrypted alphabet of key 5 that rotates from right to left

ABCDEFGHIJKLMNOPQRSTUVWXYZ

VWXYZABCDEFGHIJKLMNOPQRSTU

Note: Although there are two types of rotation, it is common to only use the rotation from left to right. And you can see that this does not affect encryption, as the encryption of example 2 could be done using left-to-right rotation key 21.

Keyless decrypting:

We can decrypt cipher-encrypted messages using intuition and logical reasoning. All of this must be based on how words are formed in the language in which the original message is found.

We can decrypt cipher-encrypted messages using intuition and logical reasoning. All of this must be based on how words are formed in the language in which the original message is found.

Take the Portuguese language as an example, where words are formed by syllables, each syllable has a vowel (a, e, i, o, u) and each syllable usually has two or three letters. To practice, we will decrypt the JHZH message. But where can we begin?

If you count the number of letters you will see that there are four, in this way we can intuit that there is a great possibility that the word is formed by two syllables. If we separate the message into two syllables (JH-ZH) we see that the letter H repeats, which gives us the hint that H is a vowel.

Since A is a letter that appears in many words of our language, I will kick that H is equivalent to A. If this is true, the key of the cipher in this case is 7. Decrypting the message using key 7, we call the following message : CASA (home in Portuguese).

Basically, when you do not know the key, you should decrease the number of attempts using the properties of the language in your favor.
